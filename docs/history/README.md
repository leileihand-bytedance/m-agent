# M-Agent 历史文档

本目录保存已经完成、失效、被替代或仅供追溯的文档，不作为当前开发依据。

```text
history/
├── designs-and-plans/              # 已完成的设计与实施计划
├── legacy-proposals/               # 早期方案和选型资料
├── quality-reviews/                 # 已完成的质量建设、基线和人工评审
```

查找当前事实时，按以下优先级：

1. 当前代码和自动化测试。
2. `docs/development/architecture.md`、模块 README 和 Skill 文档。
3. `docs/development/TODO.md`。
4. Git 历史和本机月度开发日志。
5. 本目录中的历史资料。

`archive/` 与本目录不同：`archive/` 保存已退出运行的历史代码快照；`docs/history/` 保存历史文档。
