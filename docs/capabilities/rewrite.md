# 材料润色

## 定位

材料润色用于修改用户直接粘贴的文字，默认优化表达、结构和语气，不新增事实。业务规则唯一来源是 `skills/rewrite/SKILL.md`。

## 入口

独立企业微信入口位于 `app/rewrite_bot/`，注册表只开放 `rewrite`。当前不接收文件和网页链接。

用户可以：

- 在同一条消息中提供原文和要求。
- 先发原文，待 Bot 追问后补充修改方向。
- 基于上一版继续提出修改要求。
- 明确粘贴新正文时开始新的润色任务，不误改其他写作会话。
- “只改第二段、其他不变”等局部要求使用与直报、简报相同的公共改稿协议，未点名段落由代码恢复。

## 边界

- 默认不补充原文没有的数据、案例、政策或结论。
- 与微众银行明确相关时，可以检索少量内部信息库语料核对名称和已有标准表述。
- 信息库中的新数据、日期、荣誉或内部事实不得自动写入原文。
- 待确认原文、任务和会话与写作 Bot 隔离。
