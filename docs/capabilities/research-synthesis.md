# 综合调研整合

## 定位

综合调研整合以用户提供的现成提纲为骨架，将多部门 Word、PDF 和 PPTX 材料归入对应问题并生成 Word 初稿。业务规则唯一来源是 `skills/research_synthesis/SKILL.md`。

## 流程

1. 识别总提纲和部门材料；证据不唯一时保留任务并追问。
2. 建立材料台账，记录综合事实、来源、缺口、冲突和图片位置。
3. 按提纲问题跨部门归并，不按文件顺序逐份拼接。
4. 生成正文并校正标题编号、来源标签、遗漏主题和图片提醒。
5. 任务进入写作持久队列，后台生成采用公文基本格式的 DOCX。
6. 企业微信主动发送简短完成提示和 DOCX 初稿，并分别保存交付检查点。

## 边界

- 缺少材料或口径冲突时不能静默补齐。
- 图片不自动 OCR，也不嵌入成稿；在对应位置保留人工评估提醒。
- 当前不支持基于成稿继续多轮改稿。
- 已接入写作持久任务执行器；同一消息幂等、进程恢复和逐项交付规则与其他写作长任务一致。
- 完成提示或 Word 附件已取得成功回执后，恢复任务时不会重复发送；回执无法判断时暂停自动重发并进入人工恢复。
