# 多素材简报兼容入口

`writer2` 不再维护独立业务规则、提示词或生成校验。它只用于兼容已有权限、任务卡片、写作队列和历史稿件，并把任务转发到 `skills/writer1/` 的统一简报工作流。新任务由统一 workflow 根据实际材料关系决定模式；历史 `writer2` 续改保留多素材上下文。

当前简报业务规则的唯一来源是：

```text
skills/writer1/SKILL.md
```

新增或修改简报规则时只更新统一简报 Skill，不在本目录复制规则。删除本兼容入口前，必须先完成历史任务、权限和队列类型迁移。
