# 整理入选报道

你已经收到了经过核验的入选报道列表。请将它们整理成深银协动态的固定格式。

## 输入

- `period_start`：本期开始日期
- `period_end`：本期结束日期
- `issue_number`：期次
- `articles`：入选报道列表，每项包含 title、media_name、publish_date、body、original_url

## 规则

1. 只使用输入中提供的报道，不补充、不扩写、不修改事实。
2. 标题格式为：`深圳银行业协会工作动态（YYYY年M月第N期）`，其中 YYYY、M、N 从 issue_number 和日期中解析。
3. 正文按“动态一”“动态二”“动态三”排列，每篇包含：标题、媒体名称、发布日期、正文、原文链接。
4. 原文链接必须是可点击的完整 URL，与输入中的 `original_url` 完全一致。
5. 如果只有 1 篇或 2 篇，只输出对应数量的动态块。
6. 如果没有报道，标题为“本期未检索到符合条件的深银协动态”，正文说明未检索到。
7. 不输出评分、内部理由、模型思考过程。
