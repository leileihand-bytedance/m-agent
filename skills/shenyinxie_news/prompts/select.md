# 深银协动态候选稿判断

你的任务是判断一篇外部媒体报道能否作为“微众银行正面新闻和成果”报送给深圳市银行业协会。

## 判断顺序

1. 先判断报道是否包含可核验的微众银行正面动态或成果，包括经营成果、普惠金融、科技创新、社会价值、合作成果、外部认可等。
2. 再判断微众银行在全文中的主体强度：
   - `primary`：全文唯一主体或明显主角，可原文全文采用；
   - `substantial`：综合报道中有完整、正面、可独立成立的微众银行内容，可抽取式摘编；
   - `mention`：名单、一句话或顺带提及，不能采用。
3. 其他银行为主、中性行业盘点、负面风险事件、未经证实的争议信息，返回 `reject`。

## 输出约束

- `full_text` 仅用于 `primary + is_positive_achievement=true`。
- `extract` 仅用于 `substantial + is_positive_achievement=true`。
- `extract` 的 `excerpt_paragraphs` 必须逐字复制原文中的完整段落，保持原顺序；不得概括、改写、拼句或添加衔接语。
- `suggested_title` 只在 `extract` 时填写，必须突出微众银行，但不得增加原文没有的事实、数据、评价或因果关系。
- 无法确认时选择 `reject`，不要为了凑数放宽标准。
